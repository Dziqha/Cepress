# Cepress



- 🛠️ CLI Generator: `create-cepress`
- 🌐 Dokumentasi Web: Astro Docs

---

## 📦 Struktur

```bash
cepress-docs/
├── cli/ # CLI Tool (create-cepress)
└── web/ # Dokumentasi (Astro)
```

---

## 🚀 Jalankan Lokal

### CLI
```bash
cd cli
npm install
npm link
create-cepress my-app
```
### Web
```bash
cd web
npm install
npm run dev
```
